🪲 TobaccoBug: Bio-Rearing Dashboard
A real-time data management system for rearing Nesidiocoris tenuis (predator) and Bemisia tabaci (prey).

🎯 Project Purpose
Digitization: Replace manual logs with a real-time Firebase dashboard.

Optimization: Manage the predator-prey balance between 10 main cages and backup cages.

Actionable Intelligence: Provide data-driven guidelines (e.g., "Transfer prey from Cage S9 to B3") to prevent cannibalism.

Research Ready: One-click data export for thesis and statistical analysis.
